 d["a"] = 1
 print(d["a"]) // prints 1

  • get(_:)

    Allows retrieving and strongly typing a value at the same time. This means you can avoid having to cast and unwrap values from the Document when you know what type they will be. For example:

      let d: Document = ["x": 1]
  let x: Int = try d.get("x")
  • Params:
    • key: The key under which the value you are looking up is stored
    • T: Any type conforming to the BsonValue protocol

    • Throws

    • A MongoError.typeError if the value cannot be cast to type T or is not in the Document
